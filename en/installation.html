Installation¶

Manual installation¶

RESTAS is a typical ASDF system and installing it does not differ from installing most other Common Lisp packages.

Before installing RESTAS the following dependencies need to be installed:

  • Hunchentoot

  • alexandria

  • cl-routes

After this you should get the source code. The latest version of RESTAS can be downloaded at http://code.google.com/p/restas/downloads/list.

Fans of unstable versions also can use the git repository: :

git clone git://github.com/archimag/restas.git

Now you have to unpack the source code and to add it to asdf:*central-registry*:

(push #P"/path/to/restas/" asdf:*central-registry*)

Or create a symbolic link: :

cd  /sbcl/.systems   # or any other directory, specified with asdf:*central-registry*
ln -s /path/to/restas/restas.asd .

Gentoo Linux¶

Gentoo Linux users can take advantage of archimag-lisp-overlay, a fork of gentoo-lisp-overlay.

Installation of this overlay with layman is done in the following way: :

emerge layman

Now you just need to add http://github.com/archimag/archimag-lisp-overlay/raw/master/layman-list.xml to the overlays parameter of /etc/layman/layman.cfg configuration file: :

overlays  : http://www.gentoo.org/proj/en/overlays/repositories.xml
            http://github.com/archimag/archimag-lisp-overlay/raw/master/layman-list.xml

Then: :

layman -f -a archimag-lisp
emerge restas

Optional packages¶

PURI-UNICODE¶

RESTAS depends on puri, which does not handle non-ASCII UTF-8 URLs correctly. To solve this problem you can use the puri-unicode fork of puri. The stable version of puri-unicode can be downloaded from http://github.com/archimag/puri-unicode/downloads.

SLIME-ARCHIMAG¶

SLIME, undoubtedly, is profound tool for development applications in Common Lisp, but sadly it is self-contained and doesn't provide any easy to use mechanisms of extending functionality. Meanwhile, it contains outstanding infrastructure which can be used for providing different instrumental facilities. RESTAS supports integration with SLIME and provides a series of tools simplifying web application development but for using them you need to use the SLIME-ARCHIMAG fork instead of the original SLIME and do some additional Emacs initialization: :

(add-to-list 'load-path "/path/to/restas/slime/")
(require 'restas)
